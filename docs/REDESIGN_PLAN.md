# Saarthi Redesign — Apple-glass, map-centric command center

## Context

User feedback after the dashboard critique: the radial hub feels odd as the hero; KPIs belong on the left; **the map should be ~80% of the screen** with the priority queue and live feed floating over it; cards need icons + photos/videos; add a collapsible icon sidebar, a right-side cluster detail drawer, bento analytics tiles, a social-media-style feed page, **frosted-glass Apple design language**, and **shadcn/ui**. User-confirmed decisions: **dual theme (light/dark toggle, dark default)** · **collapsible icon sidebar** · **full sweep** (replace old layout entirely) · **curated stock URLs** for media.

This rebuild also **absorbs the open critique backlog** (22/40 baseline, snapshot in `app/.impeccable/critique/`): dispatch confirm + success ceremony, Esc/focus-trap/aria (free via Radix), honest copy (make "click to filter" and "editable" true), WCAG AA contrast in both themes (current `faint` #A69C86 fails at 2.7:1), `lang="hi"` attributes, splash once-per-session + skippable, keyboard-accessible hub/queue/markers. The prior harden→clarify→polish→adapt command sequence is superseded by this redesign.

All paths relative to repo root `/Users/ian/Documents/Fun and Learning Projects/Hack2Skill - Drishti MP Goverment App`. Everything happens in `app/`; `@saarthi/shared` is untouched.

## Architecture decisions

1. **Tokens:** channel-only HSL CSS custom properties on `:root` (light) / `.dark` (dark) in `globals.css`, consumed by Tailwind as `hsl(var(--x) / <alpha-value>)`. **Keep existing semantic token names** (`canvas, surface, chip, panel, ink, body, muted, faint, line, urgency-*, pathway-*, source-*`) so existing classnames retheme for free — the Intelligence page auto-adapts with no layout work. Rename Saarthi's `accent` → `saffron` (collides with shadcn's `--accent`); restructure `muted` → `{DEFAULT: tint, foreground: text}` and migrate `text-muted` → `text-muted-foreground` (~40 hits, mechanical). Add the shadcn vocabulary (`--background, --card, --primary, --ring, --destructive`…) mapped onto Saarthi semantics. Delete `app/src/lib/ux4g-tokens.ts` (values move to globals.css; radii/fonts inline in tailwind config).
   - Light: canvas `#F4F2ED`, surface `#FFF`, ink `#14192A`, body `#4A5060`, muted-fg `#5C6270`, faint `#6B665B` (4.6:1 — fixes the AA failure), navy `#12325B`, saffron `#C15A15`.
   - Dark: canvas `#0B1120`, surface `#131C30`, ink `#EAEEF6`, body `#AAB4C6`, faint `#8B93A5`, line `#243250`; interactive navy brightens to `#7FA6E0`-family, saffron to `#E07A33`; urgency dark variants (critical `#E26A54`, high `#E0A24E`, medium `#D6C05A`, low `#84A8CB`). All body-size text ≥4.5:1 in both themes.
   - `.glass` utility: `background: hsl(var(--surface)/0.72); backdrop-filter: blur(16px) saturate(1.4); border: 1px solid hsl(var(--line)/0.6)` + `.glass-strong` (0.85) for text-dense panels. Cap ~5 blurred surfaces; never nest blurs.
2. **Theme:** `next-themes` (`attribute="class"`, `defaultTheme="dark"`, `enableSystem: false`), `suppressHydrationWarning` on `<html>`. Sun/Moon toggle in TopBar. **Leaflet theme switch without re-init:** keep `tileLayerRef`, call `setUrl(dark_all ↔ light_all)`; marker divIcon HTML uses `var(--urgency-*)` so markers retheme with zero re-render. Never key the map on theme.
3. **shadcn/ui:** pin CLI `pnpm dlx shadcn@2.3.0 init` in `app/` (last line tuned for Tailwind v3), CSS-vars mode, style "new-york". Generate: `button, badge, sheet, dialog, alert-dialog, popover, dropdown-menu, tooltip, scroll-area, separator, sonner, skeleton`. **Hand-roll the sidebar** (~120 lines; shadcn's sidebar component fights the glass aesthetic and drags in its own token family). Icons: `lucide-react`; keep bespoke `SourceIcon`/`CategoryIcon`/`BrandBars`.
   - New deps: `class-variance-authority, clsx, tailwind-merge@^2, lucide-react, next-themes@^0.4, sonner@^1.7`, the Radix primitives shadcn generates, devDep `tailwindcss-animate`. `darkMode: ["class"]`.
4. **Shell:** new `app/src/app/(dashboard)/layout.tsx`: `flex h-screen overflow-hidden` → `<Sidebar/>` (glass rail, `w-[232px] ↔ w-[64px]`, collapse state in zustand + localStorage, Tooltips when collapsed) + `<TopBar/>` (absolute glass strip: scope pill, search stub, ThemeToggle, MP avatar DropdownMenu) + children. ThemeProvider, sonner Toaster, ClusterDrawer, ActionComposer mounted here. **z-scale:** map 0 → panels 10 → topbar 30 → sidebar 40 → Sheet/Dialog 50 → toast → splash 100.
5. **Map stage (dashboard page):** wrapper `relative h-full isolate` (`isolate` contains Leaflet's internal z-1000 panes). Map `absolute inset-0 z-0`, `scrollWheelZoom: true`, zoom control bottomright. Overlay: `absolute inset-0 z-10 pointer-events-none grid grid-cols-[360px_1fr_360px] gap-4 p-4 pt-[68px]`; panels set `pointer-events-auto` — map stays draggable in the gaps. **Left column:** KpiStack (3 bento glass tiles: MPLADS, open clusters, SC/ST) + PriorityQueue glass panel (ScrollArea, flex-1). **Right column:** RadialHubTile (~260px, shrunk; spokes clickable → `sourceFilter`, keyboard-focusable) + LiveFeedPanel (flex-1, ScrollArea). Legend + visible-count as small glass chips bottom-center-left.
   - **Responsive <1024px:** drop the overlay; page scrolls normally — map as `h-[45vh]` block, panels stack below (same components, `lg:` fork). Sidebar hides; TopBar hamburger opens nav in `Sheet side="left"`.
6. **Cluster detail drawer:** shadcn `Sheet side="right"` (`sm:w-[460px]`), overlay `bg-black/20 backdrop-blur-[2px]`. Store additions: `selectedClusterId/selectCluster/closeDetail`, `sidebarCollapsed/toggleSidebar`, `sourceFilter`, `dispatched` upgraded to `{id, ref, at}[]`. All three surfaces select: queue cards become real `<button>`s, feed "joins cluster #NN" links resolve, Leaflet markers get click + `keyboard: true` + `alt`. Drawer sections: header (mono id, UrgencyBadge, trend, title + `lang="hi"` Hindi title, ward) → media strip (2–3 photos, horizontal snap scroll) → EvidenceChips → CrossRefs → suggested action or DispatchProgress → sticky footer ("Draft MPLADS letter" primary when eligible, "View on map" pans the map). 8. **/live masonry feed:** `app/src/app/(dashboard)/live/page.tsx` + `LiveMasonry` client component. **CSS columns** (`columns-1 sm:columns-2 lg:columns-3 2xl:columns-4`, cards `break-inside-avoid`), newest-first. Card: media (variable aspect) → source Badge + relative time → snippet (`lang="hi"` where flagged) → cluster link (navigates to `/dashboard` + `selectCluster` via global store). Data: extend `FeedItem` with `id/clusterId/media`, `ClusterUi` with `media?: MediaAsset[]`; new `app/src/lib/media.ts` with `MediaAsset {type, src, poster?, aspect, alt}` + 10–15 curated `images.unsplash.com/photo-<id>?w=800&q=70` direct URLs matched to categories (waterlogging/streetlights/garbage/smog/tanker) + ~16 `LIVE_FEED_ITEMS`. 2–3 video cards: poster + play glyph + duration chip; try 1–2 remote sample mp4s, degrade to poster-only. `MediaImage.tsx` wraps `next/image` with `onError` → gradient tile + CategoryIcon (offline story). `next.config.mjs`: `images.remotePatterns` for unsplash. 9. **Splash:** once per session (sessionStorage), click/Esc to skip, never input-blocking, ~2.4s.
10. **Intelligence page:** keeps its layout; auto-themes via retained token names. Only swap hardcoded hexes in `OutlookChart`/`HeroInsights` to vars, wire it under the new shell (drop AppHeader/AppFooter usage).

## Risks & mitigations

- **backdrop-filter over panning Leaflet janks:** ≤5 blurred surfaces, blur(16px) max, `translateZ(0)` on `.glass`; fallback: opacity 0.88 + blur 8px.
- **Leaflet z-1000 panes over drawer:** `isolate` on map-stage wrapper.
- **shadcn CLI emitting v4-style CSS:** pin `shadcn@2.3.0`; verify `@layer base` vars (not `@theme`) after init.
- **Unsplash/CARTO offline at demo:** MediaImage error fallback; pre-demo cache visit.
- **`text-muted` rename churn:** mechanical replace + typecheck per commit.
- **Radix Dialog over Sheet:** supported; keep both `modal`.
